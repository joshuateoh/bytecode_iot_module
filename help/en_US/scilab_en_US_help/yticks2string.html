yticks2string

Change the ytick to string

Syntax

yticks2string(str,n)

Parameters

str :

string input to replace the current yticks

n :

number of ticks in y axis

Description

This is to change the yticks to string replacing numbers

Examples

x = 1:10
plot(x,y);
xx = ['a','b','c','d','e','f','g','h','i','j','k'];
yticks2string(xx,5);
