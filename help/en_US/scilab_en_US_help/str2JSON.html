str2JSON

Convert Scilab string to JSON format

Syntax

vec = [3 2 1];
str = vec2str(vec);
out = str2JSON(str,'data');

Parameters

m :

Input string

n :

Name of object

stringJSON :

Output in JSON format

Description

List the variables available in Ubidots source and return the information of them.
