xticks2string

Change the xtick to string

Syntax

xticks2string(str,n)

Parameters

str :

string input to replace the current xticks

n :

number of ticks in x axis

Description

This is to change the xticks to string replacing numbers

Examples

x = 1:10
plot(x,y);
xx = ['a','b','c','d','e','f','g','h','i','j','k'];
xticks2string(xx,5);
